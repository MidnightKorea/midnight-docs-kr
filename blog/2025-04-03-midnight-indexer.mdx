새로운 [Midnight Indexer](https://docs.midnight.network/relnotes/midnight-indexer)을 출시했습니다. Midnight 노드에서 사용자 앱까지 블록체인 데이터 흐름을 간소화하는 모듈형 고성능 인덱서입니다.

<!--truncate-->

새 Indexer은 블록 히스토리를 조회하고 데이터를 처리합니다. 쿼리, 뮤테이션, 실시간 구독을 지원하는 유연한 GraphQL API로 데이터를 제공합니다.

Rust로 작성했으며, 기존 Scala 기반 Indexer을 대체할 예정입니다. 성능이 크게 개선되었고, 로컬과 클라우드 환경 모두에서 배포가 쉬워졌습니다. 지갑과도 더 긴밀하게 통합됩니다. PostgreSQL과 SQLite을 모두 지원합니다.

## Key Improvements Over the Legacy Indexer

🦀 Rust로 재작성하여 속도, 안정성, 유지보수성 향상

🧱 모놀리식 Scala pub-sub 설계를 대체하는 모듈형 아키텍처

🤝 최신 지갑 도구와 긴밀한 통합 (SDK v4+ 및 Lace Wallet v2.0.0+)

🌐 실시간 업데이트를 위한 구독 기능이 포함된 유연한 GraphQL API

Midnight에서 개발 중이라면, 지금 통합 테스트를 시작하고 마이그레이션을 준비하세요. Scala 기반 Pub-Sub Indexer은 당분간 작동하지만, 공식 지원이 중단(deprecated)되었으며 향후 릴리스에서 제거됩니다.

📢 지원 중단 공지

Scala 기반 Pub-Sub Indexer은 당분간 사용할 수 있지만, 공식 지원이 중단되었으며 향후 릴리스에서 제거됩니다. Midnight에서 개발 중이라면, 지금 바로 새 Indexer을 테스트하고 마이그레이션을 준비하세요.
