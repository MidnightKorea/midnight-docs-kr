Midnight 테스트넷 업그레이드로 BLS12-381이 도입되며, 개발자는 Lace 지갑, proof server을 업데이트하고 최신 Compact 컴파일러로 컨트랙트를 재배포해야 합니다.

<!--truncate-->

Midnight 테스트넷이 [업그레이드](https://midnight.network/blog/upcoming-testnet-02-upgrade-all-you-need-to-know) 중입니다. 이 업그레이드의 일환으로, 증명 시스템이 Pluto-Eris에서 BLS12-381로 변경됩니다.

# Developer requirements

호환성과 기능성을 보장하기 위해, DApp 개발자는 다음 단계를 완료해야 합니다:

- [Developer requirements](#developer-requirements)
  - [1- Installing the Lace wallet](#1--installing-the-lace-wallet)
  - [2- Downloading the proof server](#2--downloading-the-proof-server)
  - [3- Updating the Compact compiler and redeploying the smart contracts](#3--updating-the-compact-compiler-and-redeploying-the-smart-contracts)

## 1- Installing the Lace wallet
Lace 지갑의 최신 버전(2.0.0)은 [여기](https://docs.midnight.network/relnotes/lace)에서 받을 수 있습니다. 새로운 Bech32m 주소 형식을 지원합니다.

1. ZIP 파일을 다운로드하고 압축을 해제하세요.
2. Chrome 또는 Brave 브라우저를 여세요.
3. **Extensions** 패널로 이동하세요.
4. **Load unpacked**을 클릭하세요.
5. 방금 압축 해제한 폴더를 선택하세요.
6. 설치가 완료되면, 새 지갑을 생성하거나 저장된 시드 문구를 사용하여 이전 지갑을 복원할 수 있습니다.

완료되면, "mn_shield-addr_test..."로 시작하는 새로운 주소 형식이 보일 것입니다.

## 2- Downloading the proof server
proof server의 최신 버전은 [여기](https://docs.midnight.network/relnotes/proof-server)에서 다운로드할 수 있습니다.
커맨드 라인으로도 설치할 수 있습니다:
```
docker pull midnightnetwork/proof-server:latest
```  

다운로드 후 다음 명령으로 proof server을 시작할 수 있습니다:
```
docker run -p 6300:6300 midnightnetwork/proof-server -- midnight-proof-server --network testnet
```  

자세한 내용은 [문서](https://docs.midnight.network/develop/tutorial/using/proof-server)에서 확인할 수 있습니다.

## 3- Updating the Compact compiler and redeploying the smart contracts
스마트 컨트랙트는 새로운 증명 시스템과 그것이 생성하는 영지식 증명에 맞추어 재배포되어야 합니다.

이번 업그레이드의 일환으로, 컴파일러, Compact 런타임, 온체인 런타임이 모두 변경됩니다. 개발자는 새로운 BLS 호환 컴파일러 버전 0.23.0으로 컨트랙트를 재컴파일해야 하며, 이 버전은 다른 업그레이드 컴포넌트와 함께 출시될 예정입니다.

재컴파일 후, 개발자는 새로 컴파일된 컨트랙트를 사용하여 DApp을 다시 빌드해야 합니다. 또한 컨트랙트는 내보낸 회로에 대해 새로운 prover와 verifier 키가 필요합니다. 기존 키로 재배포하는 것만으로는 충분하지 않으며, 새 키를 생성하기 위해 재컴파일이 필요합니다.

기존에 사용하던 방법으로 컨트랙트를 배포할 수 있으며, 또는 [이 예시](https://github.com/midnightntwrk/example-counter/blob/1babbcb377317ba705c26545a6808deea7a39975/counter-cli/src/api.ts#L90)와 deploy 함수를 참고하여 Midnight 테스트넷에 스마트 컨트랙트를 배포할 수 있습니다.

Bech32m 주소를 지원하는 [최신 버전의 MidnightJS](https://docs.midnight.network/relnotes/midnight-js)를 사용하여 컨트랙트를 배포하세요.
