# Neovim setup for Compact

[compact.vim](https://github.com/1NickPappas/compact.vim)은 Neovim에서 Compact 언어를 지원하는 커뮤니티 플러그인입니다.

## Features

이 플러그인은 Compact 소스 파일 작업 시 다음 기능을 제공합니다:

* 구문 강조 (정규식 기반 및 tree-sitter)
* 스마트 들여쓰기
* 코드 폴딩
* 텍스트 객체 ([nvim-treesitter-textobjects](https://github.com/nvim-treesitter/nvim-treesitter-textobjects) 필요)
* `gf`를 통한 import 탐색
* `:make`를 통한 컴파일러 연동
* 로컬 스코핑 (circuit/block 단위 변수 참조)

![구문 강조](/img/neovim-setup/compact-syntax.png)

## Installation

[lazy.nvim](https://github.com/folke/lazy.nvim)으로 설치하세요:

```lua
{ "1NickPappas/compact.vim" }
```
