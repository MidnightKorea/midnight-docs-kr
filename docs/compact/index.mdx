# The Compact language

Midnight은 자체 프로그래밍 언어인 Compact를 제공합니다. Compact로 Midnight 생태계를 위한 스마트 컨트랙트를 작성할 수 있으며, Compact 컴파일러는 ledger와의 상호작용이 올바르게 이루어졌음을 증명하는 영지식 circuit를 생성합니다.

이 섹션에서는 Compact 작성법과 관련 도구 사용법을 자세히 설명합니다. 이후 섹션에서는 보안 모범 사례, Compact 언어 레퍼런스, Compact 표준 라이브러리 API 레퍼런스를 다룹니다.
