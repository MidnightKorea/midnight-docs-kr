# Reference documentation

이 섹션은 Compact 언어, TypeScript API, Midnight 팀이 제공하는 도구에 대한 레퍼런스 자료입니다. 다음 내용을 포함합니다:

- [Compact 언어](/compact): [스마트 컨트랙트 작성 튜토리얼](/compact/writing)을 포함한 Compact 사용법을 상세히 설명합니다.
- [Midnight API](/develop/reference/midnight-api): Midnight 개발에 사용할 수 있는 다양한 라이브러리의 API를 설명합니다.
- [도구](/develop/reference/tools): Compact용 Visual Studio Code 확장 프로그램과 Compact 컴파일러 매뉴얼 페이지 등 개발자 도구의 사용법을 안내합니다.
