Midnight은 zero-knowledge proof를 사용하여 contract 로직을 검증하면서도 민감한 데이터를 비공개로 유지합니다. smart contract는 공개 및 비공개 ledger에서 작동하여 transaction 상관관계를 줄이고, token 및 메타데이터의 안전한 온체인 atomic swap을 지원합니다.
