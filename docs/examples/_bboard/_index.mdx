# Create a new DApp

이 섹션에서 배울 내용:

- 공개 데이터와 비공개 데이터를 모두 다루는 컨트랙트 작성
- DApp 코드의 빠진 부분을 채워 완전히 작동하는 DApp 완성
- 테스트를 위해 Midnight Testnet과 *연결이 끊어진* Midnight 노드 실행
- 테스트 구성에서 프로덕션 구성으로 전환하여 Testnet에서 DApp 사용
