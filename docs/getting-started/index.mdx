# Get started

Midnight은 프라이버시를 보장하는 <GlossaryTerm term="DApps" />를 구축하기 위한 <GlossaryTerm term="blockchain" /> 플랫폼입니다. <GlossaryTerm term="zero-knowledge (ZK) proofs" />와 프로그래밍 가능한 기밀성 제어를 활용하여 데이터의 격리, 검증, 공유 방식을 개발자가 직접 정의할 수 있습니다.

아래 퀵스타트 워크플로우를 따라 로컬 환경을 설정하고, Midnight 네트워크에 첫 번째 애플리케이션을 배포해 보세요. Toolchain 설치, Midnight 앱 생성, <GlossaryTerm term="smart contract" /> 배포, 배포된 contract와 상호작용하는 방법까지 전 과정을 안내합니다.

이 워크플로우를 마치면 로컬 Devnet에서 동작하는 Midnight 앱을 직접 만들어 볼 수 있고, 프라이버시 기반 contract의 개발·테스트·반복 과정을 익힐 수 있습니다.
