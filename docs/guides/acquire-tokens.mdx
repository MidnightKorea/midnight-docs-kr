# Get faucet tokens

프로덕션 Midnight 네트워크에서 DApp을 실행하려면 지갑에 DUST를 보유하고 트랜잭션마다 소비해야 합니다.
Midnight DApp을 테스트하려는 개발자를 위해 무료 tDUST가 제공됩니다.

:::tip
로컬 undeployed 네트워크를 사용하는 경우 [로컬 네트워크 faucet](./midnight-local-network#get-faucet-tokens)으로 지갑에 자금을 넣고 tDUST를 생성하세요.
:::

## Get tNIGHT

tNIGHT는 Midnight 네트워크의 테스트 token으로, tDUST 생성에 사용됩니다. 다음 단계로 tNIGHT를 받으세요:

1. Lace Midnight 지갑에 로그인하고 Unshielded 지갑 주소를 복사하세요.

  :::note Bech32m 주소 형식
  모든 지갑 주소는 기본적으로 Bech32m 형식으로 표시됩니다. Faucet은 두 형식을 모두 지원하지만, 레거시(HEX) 형식은 향후 버전에서 제거될 예정입니다.
  :::

2. [Midnight Preprod faucet](https://faucet.preprod.midnight.network/)에 접속하여 Unshielded 지갑 주소를 입력하세요.

![Midnight Preprod faucet](./img/request-tokens.png)

3. **Request tokens**를 클릭하세요. 처리에 몇 초가 걸리며, 다음과 같은 트랜잭션 해시가 반환됩니다:

```
Transaction submitted. You will receive 1000 tNight, this should take about 2 mins. Its id is 00ec3ff48b7...
```

Faucet에서 1000.0 tNIGHT가 입금됩니다. 지갑에서 잔액을 확인하세요.

## Delegate

tNIGHT를 받은 후 tDUST 생성을 위해 위임해야 합니다. Lace 지갑에서의 절차:

1. Lace 지갑에 로그인하고 **Generate tDUST**를 클릭하세요. 입력 필드에 tDUST 주소가 자동으로 채워집니다.

![tDUST 생성](./img/delegate-dust.png)

2. **Review transaction**을 클릭한 후 **Confirm**으로 트랜잭션을 제출하세요.

![트랜잭션 검토](./img/review-transaction.png)

tDUST 탱크가 지갑용 테스트 token을 생성하는 것을 확인할 수 있습니다.

![트랜잭션 확인](./img/dust-tank-generation.png)

## Troubleshooting

Faucet 사용 시 자주 발생하는 문제와 해결 방법입니다.

### Provided address is invalid

- **오류**: `Provided address is invalid`

- **해결 방법**: 올바른 주소인지 확인하세요. Lace Midnight 지갑의 Unshielded 주소여야 합니다.

### Request limit exceeded

- **오류**: `{"status":"rate_limit_error","message":"Reached maximum number of requests for this address"}`

- **해결 방법**: 몇 시간 후 다시 시도하세요. 24시간이 지나도 해결되지 않으면 [Midnight 지원](mailto:support@midnight.network)에 문의하거나 [Midnight Discord](https://discord.gg/midnightnetwork)에서 도움을 요청하세요.
