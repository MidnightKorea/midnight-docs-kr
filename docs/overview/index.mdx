## Midnight Network에 오신 것을 환영합니다

Midnight은 공개 검증 가능성과 기밀 데이터 처리를 결합한 프라이버시 우선 블록체인(privacy-first blockchain)입니다. Midnight의 <GlossaryTerm term="selective disclosure" />와 <GlossaryTerm term="ZK proofs" />를 사용하여 다음과 같은 앱을 구축할 수 있습니다:

- 민감한 데이터를 공개하지 않고도 검증 가능한 정확성
- 사용자가 공개하기로 선택한 정보만 공유 가능
- 비공개 기록의 기밀성을 유지하면서 증명하는 규제 준수성

Midnight의 <GlossaryTerm term="consensus mechanism" />은 프라이버시와 성능을 확보함과 동시에 상태에 대한 합의를 보장합니다.
