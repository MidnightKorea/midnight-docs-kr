# Compact.js

Compact.js은 Compact 컴파일러로 컴파일된 스마트 컨트랙트를 위한 TypeScript 기반 실행 환경을 제공합니다.
Compact 스마트 컨트랙트을 컴파일하면 JavaScript 파일과 TypeScript 선언 파일이 출력됩니다.

Compact.js은 런타임에 이 파일들을 사용하여 회로를 실행합니다.
회로 실행 결과는 Midnight.js와 같은 상위 도구 및 프레임워크에서 트랜잭션을 생성하고 Midnight 블록체인에 제출하는 데 사용됩니다.

자세한 내용은 [Compact 언어 문서](../compact)를 참고하세요.
