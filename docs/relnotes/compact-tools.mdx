# Compact developer tools

Compact은 Midnight의 전용 스마트 컨트랙트 프로그래밍 언어로, 안전하고 효율적이며 적응력 있는 탈중앙화 애플리케이션을 구축하기 위해 설계되었습니다.

Compact developer tools은 Compact 툴체인과 컴파일러를 설치, 업데이트, 관리 및 실행하기 위한 명령줄 유틸리티입니다.

[관련 문서 링크](../compact/)

컴포넌트를 다운로드하려면 **Artifacts** 아래의 해당 링크를 클릭하세요.
