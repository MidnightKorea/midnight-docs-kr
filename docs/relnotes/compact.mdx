# Compact toolchain

Compact은 Midnight의 스마트 컨트랙트 프로그래밍 언어로, 안전하고 효율적이며 유연한 탈중앙화 애플리케이션을 구축하기 위해 설계되었습니다.

Compact 툴체인은 Compact 컴파일러, 포매터, 그리고 Compact "fixup" 도구를 포함합니다.
컴파일러는 Compact 스마트 컨트랙트 코드를 JavaScript 구현체와 영지식 회로 표현으로 변환합니다.

[Link to related documentation](../compact/compilation-and-tooling/compiler-usage)

툴체인을 설치하려면 먼저 [compact devtools](compact-tools.mdx)을 설치한 다음, `compact update`(최신 버전용) 또는 `compact update 0.28.0`(특정 버전, 예: 0.28.0 설치용)을 사용하세요.
