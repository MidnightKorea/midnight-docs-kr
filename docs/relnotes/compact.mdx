# Compact toolchain

Compact은 Midnight 전용 스마트 컨트랙트 언어로, 안전하고 효율적인 탈중앙화 애플리케이션 개발을 위해 만들어졌습니다.

Compact 툴체인에는 컴파일러, 포매터, "fixup" 도구가 포함되어 있습니다.
컴파일러는 Compact 스마트 컨트랙트 코드를 JavaScript 구현체와 영지식 회로로 변환합니다.

[Link to related documentation](../compact/compilation-and-tooling/compiler-usage)

툴체인을 설치하려면 먼저 [compact devtools](compact-tools.mdx)을 설치하세요. 이후 `compact update`(최신 버전) 또는 `compact update 0.28.0`(특정 버전 지정)으로 설치할 수 있습니다.
