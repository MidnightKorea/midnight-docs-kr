# DApp Connector API

DApp Connector API는 탈중앙화 애플리케이션(DApp)에서 지갑 접근 요청, 인가 확인, Midnight 네트워크 연동에 필요한 서비스 URI 조회 기능을 제공합니다.

[관련 문서 링크](/api-reference/dapp-connector/)
