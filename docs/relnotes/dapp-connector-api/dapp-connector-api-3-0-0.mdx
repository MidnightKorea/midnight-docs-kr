# DApp Connector API

DApp Connector API은 탈중앙화 애플리케이션(DApp)이 지갑 접근을 요청하고, 인가를 확인하며, Midnight 네트워크와 상호작용하기 위한 서비스 URI를 조회할 수 있도록 합니다.

자세한 내용은 [DApp connector 문서](../../develop/reference/midnight-api/dapp-connector/interfaces/DAppConnectorAPI)를 참고하세요.
