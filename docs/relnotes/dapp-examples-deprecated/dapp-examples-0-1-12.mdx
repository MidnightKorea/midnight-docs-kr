# Midnight Example Applications

Midnight 예제 애플리케이션은 다양한 사용 사례와 스마트 컨트랙트 기능을 보여줍니다.

- Bulletin Board DApp (`bboard`) -- 사용자가 메시지를 게시하고 삭제할 수 있는 간단한 단일 항목 게시판입니다. 튜토리얼 버전(`bboard-tutorial`)이 포함되어 있어, 개발자가 누락된 컴포넌트를 직접 구현한 뒤 완성된 솔루션을 검토할 수 있습니다.
- Counter DApp (`counter`) -- 공개적으로 저장된 카운터를 유지하며, 사용자가 값을 증가시킬 수 있는 스마트 컨트랙트입니다. 이 예제는 Midnight의 도구를 사용한 컨트랙트 개발, 테스트, 배포를 다룹니다.
