# Midnight Indexer

Midnight Indexer는 Midnight 노드의 블록체인 데이터를 최종 사용자 애플리케이션에 효율적으로 전달하는 컴포넌트입니다. 블록 이력 조회 및 처리, 인덱싱 데이터 저장, GraphQL API를 통한 쿼리 및 구독 기능을 제공합니다.

컴포넌트를 다운로드하려면 **Artifacts** 아래의 해당 링크를 클릭하세요.
