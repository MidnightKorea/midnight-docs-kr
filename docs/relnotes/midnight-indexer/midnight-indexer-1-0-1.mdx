# Midnight Indexer

Midnight Indexer는 Midnight 노드에서 최종 사용자 애플리케이션으로 블록체인 데이터 흐름을 최적화하기 위해 설계된 컴포넌트 세트입니다. 블록 이력을 조회하고 처리하며, 인덱싱된 데이터를 효율적으로 저장하고, 쿼리와 구독을 위한 GraphQL API를 제공합니다.

## Midnight Indexer 1.0.1 Release Notes

### Stable Release

* **Version**: 1.0.1
* **Audience**: 노드 운영자, Midnight 생태계와 통합하는 개발자
* **Environment**: 프로덕션 준비 완료

이번 릴리스(1.0.1)는 안정적이며 프로덕션 환경에 배포할 수 있습니다. 공개 및 프로덕션 환경에서 안전하게 사용할 수 있습니다.

### Key Changes & Highlights

:::note
이 릴리스는 wallet-sdk v4.0.0 이상(Lace Wallet v2.0.0 이상과 호환)이 필요합니다.
:::

### Changelog

#### Bug Fixes

* 활성 지갑을 명시적으로 표시하여 의도하지 않은 연결 해제를 방지하도록 지갑 세션 관리를 개선했습니다.
* 지갑 구독이 지갑의 실제 알려진 인덱스와 관계없이 항상 인덱스 0부터 진행 업데이트를 다시 시작하는 문제를 수정했습니다.

### Breaking Changes

#### Migration from Scala-based to Rust-based indexer

**영향**: API 엔드포인트는 유사하나, 내부 구현 및 성능 특성이 다릅니다
**필요한 조치**:

* 새로운 Rust 기반 인덱서에서 통합을 테스트하세요: `https://indexer-rs.testnet-02.midnight.network/api/v1/graphql`
* 이전 엔드포인트(`indexer.testnet-02...`)에서 마이그레이션을 시작하세요
* 향후 릴리스에서 Scala 인덱서의 지원 종료를 모니터링하세요

### Known Issues

* **macOS 호환성**: macOS에서의 테스트가 제한적입니다. 네이티브 SQLite 통합에서 문제가 발생할 수 있습니다.
* **Windows 지원**: 동작하지만 공식적으로 지원되지는 않습니다. Linux 환경을 권장합니다.

### Compatibility

* **운영 체제**: Linux에서 완전 지원됩니다. macOS와 Windows는 제한적으로 지원됩니다.
* **서드파티 도구**: 모니터링 및 분석 플랫폼과 호환됩니다. 약간의 설정 변경이 필요할 수 있습니다.
* **Midnight 생태계**: Midnight 노드 API 및 관련 인프라와 완전히 호환됩니다.

### Additional Resources

* **커뮤니티 지원**: [Discord](https://discord.com/invite/midnightnetwork)에서 논의에 참여하세요

### Upgrade/Migration Guide

#### Prerequisites

* PostgreSQL 설치 및 구성 완료(프로덕션용)
* Midnight 노드에 대한 네트워크 접근

### Installation & Migration:

* 새 인덱서 바이너리를 배포하세요
* 적절한 백엔드(SQLite 또는 PostgreSQL)에 연결하세요
* API 통합을 새 엔드포인트로 변경하세요

###  Post-Upgrade Checks

* GraphQL 쿼리가 예상대로 처리되는지 확인하세요
* 로그에서 오류 없이 시작되는지 모니터링하세요
