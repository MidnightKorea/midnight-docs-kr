# Midnight.js

Midnight.js은 Midnight 블록체인에서 탈중앙화 애플리케이션 개발을 지원하기 위해 설계된 클라이언트 라이브러리입니다.

[관련 문서 링크](/api-reference/midnight-js/packages)

컴포넌트를 다운로드하려면 **Artifacts** 아래의 해당 링크를 클릭하세요.
