## Midnight.js 2.0.0 Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight.js의 최신 업데이트와 개선 내용을 안내합니다.

### Key Changes & Highlights


- - typedoc.json 업데이트
- 깨진 compact 패키지 수정
- docs 워크플로 수정
- API 문서 수정
- docs 워크플로 재수정
- 의존성 cross-fetch을 v4.1.0으로 업데이트
- 의존성 node을 v22.14.0으로 업데이트
- 의존성 고정
- 의존성 graphql을 v16.10.0으로 업데이트
- devDependencies (비메이저) 업데이트
- 의존성 @rollup/plugin-node-resolve을 v16으로 업데이트
- 의존성 @apollo/client을 v3.13.6으로 업데이트
- 의존성 graphql-ws을 v6으로 업데이트
- 의존성 express을 v5로 업데이트
- API 문서 업데이트
- 의존성 node을 v22.15.0으로 업데이트
- BLS 지원 추가
- GraphQL 스키마에서 `contract`을 `contractAction`으로 이름 변경
- 릴리스 2.0.0-rc.1
- yarn을 v4.9.1로 업데이트
- @apollo/client을 v3.13.8로 업그레이드하고 CJS 모듈만 강제 임포트
- 릴리스 2.0.0-rc.2
- Node.js 호환성을 위한 단일 테스트 추가
- 릴리스 2.0.0 준비
