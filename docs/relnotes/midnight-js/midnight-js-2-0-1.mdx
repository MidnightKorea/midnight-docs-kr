## Midnight.js 2.0.1 Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight.js의 최신 업데이트와 개선 내용을 안내합니다.

### Key Changes & Highlights

- `EncPublicKey`을 Bech32m 형식 문자열로 처리하도록 변경
- zSwap-utils의 `createUnprovenOutput`이 segment 0을 사용하도록 업데이트
