## Midnight.js 2.1.0 Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight.js의 최신 업데이트와 개선 내용을 안내합니다.

### Key Changes & Highlights

#### Features

- 블록 타임 지원 추가

#### Notable Updates

- Compact 컴파일러을 v0.26.0으로 업그레이드
- Apollo Client v3.13.9
- 보안 및 안정성을 위한 주요 의존성 업데이트
- Compact-runtime을 0.9.0으로 업그레이드

#### Security

- 업데이트된 권한으로 GitHub Actions 보안 강화
- 의존성 전반에 보안 패치 적용

#### Developer Tooling

- ESLint 설정 최적화
- 종합 문서를 포함한 E2E 테스트 프레임워크 개선
