## Midnight Wallet API 3.5.0 Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight Wallet API의 최신 업데이트와 개선 내용을 안내합니다.

- 상태에서 코인 nullifier을 노출합니다.
