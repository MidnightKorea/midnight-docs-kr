# Midnight Wallet API

Midnight Wallet API은 지갑 운영을 위한 포괄적인 인터페이스를 제공하며, 지갑 상태의 구조, 상태와 상호작용하는 메서드, 내부에서 사용되는 타입과 변수를 정의합니다.

## Midnight Wallet API 3.6.0 Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight Wallet API의 최신 업데이트와 개선 내용을 안내합니다.

#### New Features  

- Bech32 주소를 위한 추가 필드를 노출했습니다.  
- API 문서를 업데이트했습니다.  
- ZSwap을 버전 3.0.3으로 업데이트한 후 3.0.2로 롤백했습니다.  
- **nixpkgs**을 **nixos-24.11**로 업데이트했습니다.  
- SRE 인스턴스를 위한 **renovate.json**을 업데이트했습니다.  

#### Bug Fixes  

- Docusaurus 문자 호환성 이슈를 수정했습니다.  
- 공개/비공개 README 불일치를 수정했습니다.
