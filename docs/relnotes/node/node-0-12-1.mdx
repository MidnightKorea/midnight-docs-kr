## Midnight Node 0.12.1 Release Notes

### Stable Release

* **Version**: 0.12.1
* **Audience**: 노드 운영자, Midnight 생태계와 통합하는 개발자
* **Environment**: 프로덕션 준비 완료

이번 릴리스(0.12.1)은 안정적이며 프로덕션에 사용할 수 있습니다. 공개 환경 및 프로덕션 환경에 안전하게 배포할 수 있습니다.

### Key Changes & Highlights

:::note
이번 릴리스는 0.12.0과 하위 호환되므로 체인 리셋이 필요하지 않습니다.
:::

* devnet 및 testnet-02 배포를 위한 새로운 **chain-spec**이 추가되었습니다.
* 설정 환경 변수(`NODE_KEY`, `SEED_PHRASE`)가 폐지되었으며, 대신 마운트된 시크릿을 사용하세요.

### Changelog

#### Added

- `testnet-02` 네트워크용 모의 Ariadne를 수정했습니다.
- 노드 시크릿에 파일을 사용하도록 했습니다(하위 호환성 유지).
- 체인 스펙 파일의 이름을 변경했습니다.
- midnight-docs에서 `tree` 명령어를 참조하도록 수정했습니다.
- 인증 키 설정에서 사용하지 않는 `ss58` 필드를 제거했습니다.

#### Changed

- 체인 스펙을 선언적으로 생성하도록 변경했습니다(더 DRY하고 오류 발생 가능성 감소).
- 체인 스펙의 이름을 `res/$CFG_PREFIX/chain-spec.json`으로 변경했습니다.
