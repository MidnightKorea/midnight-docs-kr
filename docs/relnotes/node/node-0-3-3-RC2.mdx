## Midnight 0.3.3 RC2 release notes

Midnight 노드의 최신 업데이트와 개선 사항을 안내합니다. 새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등이 포함되어 있습니다.

### New Features

- `address.json` 참조 지원을 추가했습니다.
- 더미 Ariadne를 사용하여 노드 설정 프로세스를 단순화했습니다.
- `.envrc` 내부에서 `.envrc.local`을 소싱하여 환경 구성 처리를 개선했습니다.
- 업그레이드 스크립트를 도입했습니다.
- 노드를 **Sidechains v0.0.26**으로 업데이트했습니다.

### Bug Fixes

- 제너레이터 빌드 문제를 수정했습니다.
- 풀 리퀘스트에서 이미지 빌드를 테스트하여 불일치 문제를 해결했습니다.
- 추가 RPC에 대한 E2E 테스트를 추가했습니다.
- 핫픽스 변경 사항을 백포트했습니다.

### Breaking Changes

- 체인 이름을 네트워크 ID로 대체했습니다.
- 설정에서 기본 멤버를 제거했습니다.

### Known Issues

- 일부 기능은 아직 최적화가 필요할 수 있습니다.

### Upgrade Instructions

1. **Sidechains v0.0.26** 업데이트를 풀(pull)하여 노드를 최신 버전으로 업데이트하세요.
2. 사용자 정의 환경 변수를 위해 `.envrc` 파일이 `.envrc.local`을 참조하도록 설정하세요.
