## Midnight 0.3.5 RC1 Plonk-only release notes

Midnight 노드의 최신 업데이트와 개선 사항을 안내합니다. 새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등이 포함되어 있습니다.

### New Features

- 빌드 단순화를 도입했습니다.
- 디버깅 파드용 슬립 모드를 추가했습니다.
- `ariadne-qa/qanet`용 `addresses.json`을 업데이트했습니다.
- devnet 및 qanet용 체인 스펙을 업데이트했습니다.
- devnet 파라미터를 재생성했습니다.

### Bug Fixes

- 오래된 `inputoutput` 이미지 참조를 수정했습니다.
- `cargo.lock`을 업데이트했습니다.
- 체인 스펙 파일 및 `addresses.json` 업데이트 관련 문제를 해결했습니다.

### Breaking Changes

- 추가 역직렬화 검사를 제거하여 트랜잭션 처리 속도를 향상시켰습니다.

### Known Issues

- 이전 체인 스펙을 업데이트된 버전으로 마이그레이션할 때 사소한 불일치가 발생할 수 있습니다.

### Upgrade Instructions

1. 빌드 단순화가 포함된 최신 업데이트를 풀(pull)하여 빌드 프로세스를 단순화하세요.
2. `ariadne-qa/qanet`의 최신 버전에 맞게 `addresses.json`을 업데이트하세요.
3. 새로운 체인 스펙 및 설정에 맞추어 devnet 파라미터를 재생성하세요.
4. 추가 역직렬화 검사 제거에 맞게 트랜잭션 처리 워크플로가 적응되었는지 확인하세요.
