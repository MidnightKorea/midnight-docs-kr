## Midnight 0.6.4 testnet release notes

Midnight 노드의 최신 업데이트와 개선 사항을 안내합니다. 새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등이 포함되어 있습니다.

### New Features

- 최대 할당 한도를 4배로 증가시켰습니다.
- 트랜잭션 검증 속도를 개선했습니다.
- 네이티브 플랫폼에서만 원장 상태에 접근할 수 있도록 하여 **testnet v1.1**의 호환성과 보안을 향상시켰습니다.

### Bug Fixes

- testnet 요구 사항에 맞추기 위해 비네이티브 플랫폼에서의 원장 상태 접근 문제를 수정했습니다.
- testnet 관련 수정 사항을 반영하여 노드 이미지를 업데이트했습니다.

### Breaking Changes

- testnet 원장 상태 동작을 업데이트했습니다.

### Known Issues

- 리소스 집약적인 시나리오에서 트랜잭션 검증에 대한 추가 최적화가 필요할 수 있습니다.

### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.6.4** 버전으로 업데이트하세요.
2. 새로운 최대 할당 한도가 설정에 반영되었는지 확인하세요.
3. 트랜잭션 검증 워크플로를 테스트하세요.
4. testnet v1.1 업데이트 준수를 위해 네이티브 플랫폼에서만 원장 상태에 접근되는지 확인하세요.
