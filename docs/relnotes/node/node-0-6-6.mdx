## Midnight 0.6.6 testnet release notes

다가오는 Cardano 하드포크에 대비하기 위해 마련된 긴급 릴리스입니다.

### New Features

- testnet을 **partnerchains 1.2.0**으로 업데이트했습니다.
- testnet 브랜치의 열거형 인덱스를 수정했습니다.

### Bug Fixes

- testnet 브랜치의 잘못된 열거형 인덱스 문제를 해결했습니다.

### Breaking Changes

- 이번 릴리스는 **partnerchains 1.2.0**에 맞추기 위해 testnet 설정을 업데이트하므로, 사용자는 해당 변경 사항과 환경이 호환되는지 확인해야 합니다.

### Known Issues

- 하드포크 전환 과정에서 통합을 보장하기 위한 추가 테스트가 필요할 수 있습니다.

### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.6.6** 버전으로 업데이트하세요.
2. testnet 환경이 **partnerchains 1.2.0**과 호환되는지 확인하세요.
3. 수정된 인덱스와의 호환성을 보장하기 위해 업데이트된 열거형과 관련된 워크플로를 테스트하세요.
