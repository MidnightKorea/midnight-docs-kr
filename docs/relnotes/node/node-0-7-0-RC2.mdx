## Midnight 0.7.0 RC2 release notes

Midnight 노드의 최신 업데이트와 개선 사항을 안내합니다. 새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등이 포함되어 있습니다.

### New Features

- 업그레이드 투표 임계값 설정을 노출했습니다.
- 노드 이미지 메모리 프로파일링 기능을 추가했습니다.
- partnerchains-dev 설정을 게시하는 워크플로를 도입했습니다.
- `+generate-qanet-keys` 빌드 옵션을 통해 `qanet`용 키를 생성하는 스크립트를 추가했습니다.
- 체인 스펙에 알려진 부트노드를 추가했습니다.

### Bug Fixes

- mint coins 호스트 함수를 수정했습니다.
- 트랜잭션 검증 시 메모리 사용량을 줄였습니다.
- `MidnightRuntimeApi::get_decoded_transactions()` 엔드포인트 버전을 올렸습니다.
- 워크플로의 사소한 스크립트 문제를 해결했습니다.

### Breaking Changes

- Git 태깅 및 GitHub 릴리스 자동화를 포함한 릴리스 워크플로를 변경했습니다.
- 알려진 부트노드가 포함된 체인 스펙 업데이트로 인해 기존 설정을 호환성에 맞게 업데이트해야 할 수 있습니다.

### Known Issues

- 새로운 업그레이드 투표 임계값 기능과 관련된 워크플로에 대한 추가 테스트가 필요할 수 있습니다.

### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.7.0-rc2** 버전으로 업데이트하세요.
2. 체인 스펙의 새로운 부트노드를 통합하도록 설정을 조정하세요.
3. 트랜잭션 검증 워크플로를 테스트하세요.
4. 새로운 릴리스 자동화 프로세스를 반영하여 통합 워크플로를 업데이트하세요.
