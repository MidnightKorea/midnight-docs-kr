## Midnight 0.7.0 RC3 release notes

Midnight 노드의 최신 업데이트와 개선 사항을 안내합니다. 새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등이 포함되어 있습니다.

### New Features

- **MidnightRuntimeApi**에 대한 지원을 강화하여 RPC 엔드포인트가 여러 런타임 API 버전을 처리할 수 있도록 했습니다.

### Bug Fixes

- RPC 엔드포인트가 호환성 문제나 런타임 오류 없이 여러 `MidnightRuntimeApi` 버전을 처리할 수 있도록 문제를 수정했습니다.

### Breaking Changes

- RPC 처리 변경으로 인해 이전 버전의 `MidnightRuntimeApi`와 상호 작용하는 도구 및 워크플로를 조정해야 할 수 있습니다.

### Known Issues

- 사용자 정의 런타임 API를 사용하는 환경에서 추가 테스트가 필요할 수 있습니다.

### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.7.0-rc3** 버전으로 업데이트하세요.
2. 향상된 런타임 API 처리와의 호환성을 보장하기 위해 RPC 엔드포인트에 의존하는 모든 워크플로를 테스트하세요.
