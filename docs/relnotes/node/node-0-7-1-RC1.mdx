## Midnight 0.7.1 RC1 release notes

이번 패치 릴리스는 Midnight 노드의 안정성과 효율성을 개선하기 위한 성능 최적화, 캐싱 개선, CI 수정에 집중합니다.

### New Features

- 상태 관리를 위한 LRU 캐시 크기를 확대했습니다.

### Changes

- Docker 이미지를 다시 정상적으로 게시하도록 CI(지속적 통합) 파이프라인을 수정했습니다.
- 런타임 `spec_version`을 `000_007_001`로 업데이트했습니다.
- LRU 캐시 초기화 값을 조정했습니다.

### Bug Fixes

- 새로운 이미지의 올바른 배포를 방해하던 CI 게시 문제를 해결했습니다.
- 증가된 워크로드 요구에 맞추기 위해 캐시 설정의 불일치를 수정했습니다.

### Breaking Changes

없음

### Known Issues

- 높은 트랜잭션 부하를 수반하는 특정 상황에서 캐싱에 대한 추가 개선이 필요할 수 있습니다.


### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.7.1 RC1** 버전으로 업데이트하세요.
2. 파이프라인에서 CI 빌드 Docker 이미지가 올바르게 배포되는지 확인하세요.
3. 고부하 환경에서 노드 성능을 테스트하여 상태 캐시 크기 증가의 효과를 확인하세요.
