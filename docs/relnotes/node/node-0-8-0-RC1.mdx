## Midnight 0.8.0 RC1 release notes

이번 릴리스는 원장 업그레이드 지원, Prometheus 메트릭 통합, 다양한 성능 최적화 및 수정 사항을 포함하여 Midnight 노드를 개선합니다.


### New Features

- Ledger 3.0.5로 업그레이드했습니다.
- 원장에 Prometheus 메트릭을 추가했습니다.
- 향후 릴리스를 위한 상세 릴리스 노트를 자동 생성하는 도구를 도입했습니다.
- `res` 디렉토리에 `qanet`용 거버넌스 키를 추가했습니다.
- `midnight-pallet`에 대한 벤치마크를 도입했습니다.


### Changes

- Rust 컴파일러를 1.82에서 1.83 버전으로 업데이트했습니다.
- 기본 캐시 크기를 0으로 설정했습니다.
- 남은 스토리지가 512MB일 때 노드를 종료하도록 기본 동작을 변경했습니다(이전에는 1024MB).
- `generate-contract-calls` 함수가 기본적으로 무작위 논스를 사용하도록 수정했습니다.
- 트랜잭션 제너레이터 JSON 파일의 직렬화/역직렬화 시 네트워크 ID에 대해 스레드 로컬 전역 변수를 사용하도록 전환했습니다.
- `qanet` 및 `local` 제네시스 설정에서 펀딩 지갑에 대한 새로운 키 유도 방법을 채택했습니다.


### Bug Fixes

- 트랜잭션 제너레이터의 논스 처리를 개선했습니다.
- 스토리지 캐시 설정 관련 문제를 해결했습니다.


### Breaking Changes

- Rust 컴파일러 요구 사항이 **1.83**으로 높아졌습니다.
- 기본 스토리지 동작 변경이 제한된 스토리지 용량의 노드에서 워크플로에 영향을 줄 수 있습니다.

### Known Issues

- 트라이 캐시 조정과 관련된 사용자 정의 설정에 대한 추가 테스트가 필요할 수 있습니다.


### Upgrade Instructions

1. 최신 변경 사항과 자산을 풀(pull)하여 노드를 **0.8.0-rc1** 버전으로 업데이트하세요.
2. 호환성을 보장하기 위해 Rust 컴파일러를 **1.83**으로 업그레이드하세요.
3. 향상된 관측 가능성과 모니터링을 위해 Prometheus 메트릭을 구성하세요.
4. 논스 처리 및 직렬화 변경 사항을 반영하여 트랜잭션 제너레이터 워크플로를 업데이트하세요.
5. 배포 환경과의 호환성을 위해 스토리지 임계값 및 캐시 설정을 검토하세요.
