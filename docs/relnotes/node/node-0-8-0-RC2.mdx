## Midnight 0.8.0 RC2 release notes

이번 프리릴리스는 향상된 트랜잭션 검증, Prometheus 메트릭, 원장 업데이트를 포함하여 Midnight 노드에 대한 추가 개선 및 수정 사항을 도입합니다.


### New Features

- 원장 측에 기본 Prometheus 메트릭을 도입했습니다.
- Midnight 팔렛에 벤치마크를 추가했습니다.
- 역직렬화된 트랜잭션을 표시하는 명령어를 추가했습니다.
- 네트워크 ID 직렬화/역직렬화를 위해 스레드 로컬 전역 변수를 사용하도록 트랜잭션 제너레이터를 리팩토링했습니다.

### Changes

- 원장을 3.0.6으로 업데이트했습니다.
- 기본 트라이 캐시 크기를 0으로 설정했습니다.
- Rust 컴파일러를 1.83 버전으로 업그레이드했습니다.
- `res` 폴더에 `qanet` 거버넌스 키를 추가했습니다.
- 필요한 설정량을 줄였습니다.

### Bug Fixes

- 트랜잭션 검증을 복원했습니다.
- 부트노드의 DNS 항목을 수정했습니다.
- 네트워크 ID를 제거하여 최신 사양에 맞게 주소 형식을 업데이트했습니다.
- 동기화 스크립트에 텔레포트 로그인을 추가했습니다.
- `generate-contract-calls` 기능이 기본적으로 무작위 논스를 사용하도록 수정했습니다.


### Breaking Changes

- Ledger 3.0.6 업데이트로 인해 `qanet` 및 미배포 제네시스 파일을 재생성해야 합니다.


### Known Issues

- 특정 상황에서 트랜잭션 역직렬화에 대한 추가 최적화가 필요할 수 있습니다.


### Upgrade Instructions

1. 새로운 노드 이미지 **midnight-node:0.8.0-e9974d3e**를 풀(pull)하고 배포하세요.
2. Ledger 3.0.6에 맞추어 `qanet` 및 미배포 제네시스 파일이 업데이트되었는지 확인하세요.
3. 원장 성능에 대해 새로 노출된 메트릭을 모니터링하도록 Prometheus를 구성하세요.
4. 단순화된 설정을 검토하고 그에 맞게 배포를 업데이트하세요.
