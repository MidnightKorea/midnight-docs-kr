## Midnight 0.8.0 RC3 release notes

이번 프리릴리스는 메트릭 개선, 키 유도 방법 향상, testnet 환경을 위한 추가 설정 등 주요 업데이트를 도입합니다.



### New Features

- 실시간 모니터링을 위해 원장에 Prometheus 메트릭을 통합했습니다.
- 성능 평가를 위해 `midnight-pallet`에 대한 종합 벤치마크를 추가했습니다.
- 관리 간소화를 위해 `res` 디렉토리에 `qanet`용 거버넌스 키를 포함시켰습니다.
- 릴리스 노트 생성을 자동화하는 도구를 도입했습니다.
- 테스트 지원 향상을 위해 `testnet-02` 설정을 노드에 추가했습니다.

### Changes

- 원장을 3.0.5로 업그레이드한 후 3.0.6으로 추가 업그레이드했습니다.
- 기본적으로 무작위 논스를 사용하도록 하여 트랜잭션 제너레이터의 `generate-contract-calls` 관련 문제를 해결했습니다.
- testnet을 제외한 모든 네트워크에서 신규 및 레거시 키 유도 방법을 구현했습니다.
- `qanet` 및 `local` 제네시스 파일에서 펀딩 지갑에 신규 키 유도를 적용하도록 업데이트했습니다.
- Rust 컴파일러를 1.82에서 1.83 버전으로 업그레이드했습니다.
- 기본 트라이 캐시 크기를 0으로 설정했습니다.
- 기본 종료 동작을 남은 스토리지 1024MB 대신 512MB에서 트리거되도록 조정했습니다.
- 네트워크 ID 직렬화/역직렬화를 위해 스레드 로컬 전역 변수를 사용하도록 트랜잭션 제너레이터를 리팩토링했습니다.


### Known Issues

- 고부하 시나리오에서의 벤치마크 추가 검증이 진행 중입니다.

### Upgrade Instructions

1. 이번 릴리스에 명시된 노드 이미지를 배포하세요.
2. Ledger 3.0.6과의 호환성을 확인하세요.
3. 새로운 `testnet-02` 설정을 검증하고 적용하세요.
4. 해당하는 경우 신규 및 레거시 키 유도 방법에 맞게 제네시스 파일을 업데이트하세요.
