## Midnight 0.8.0-rc4 Release Notes

이번 프리릴리스는 모니터링, 벤치마킹, 키 관리, testnet 설정의 개선에 집중하여 안정성과 성능을 향상시킵니다.



### New Features

- 원장 모니터링 및 인사이트를 위한 Prometheus 메트릭을 통합했습니다.
- 성능 측정 및 추적을 위해 `midnight-pallet`에 벤치마크를 도입했습니다.
- `res` 디렉토리에 `qanet`용 거버넌스 키를 추가했습니다.
- 릴리스 노트 생성을 자동화하는 도구를 구현했습니다.
- `testnet-02` 지원을 추가했습니다.

### Changes

- 원장을 3.0.5로 업그레이드한 후 3.0.6으로 추가 업그레이드했습니다.
- 기본적으로 무작위 논스를 사용하도록 트랜잭션 제너레이터의 `generate-contract-calls` 기능을 개선했습니다.
- testnet을 제외한 모든 네트워크에서 신규 및 레거시 키 유도 방법을 모두 활성화했습니다.
- `qanet` 및 `local` 제네시스 파일의 키 유도를 업데이트했습니다.
- Rust 컴파일러를 1.82에서 1.83 버전으로 업그레이드했습니다.
- 기본 트라이 캐시 크기를 0으로 설정했습니다.
- 종료 트리거를 남은 스토리지 512MB로 조정했습니다(이전에는 1024MB).
- 네트워크 ID 직렬화 및 역직렬화를 위해 스레드 로컬 전역 변수를 사용하도록 트랜잭션 제너레이터를 리팩토링했습니다.


### Known Issues

- 특정 사용 사례에 대해 벤치마크의 추가 개선이 필요할 수 있습니다.

### Upgrade Instructions

1. 새로운 기능과 개선 사항에 접근하려면 지정된 노드 이미지를 사용하세요.
2. **Ledger 3.0.6**으로 업그레이드하여 호환성을 확보하세요.
3. 확장된 테스트 역량을 위해 `testnet-02` 설정을 적용하세요.
4. 신규 및 레거시 키 유도 방법을 모두 지원하도록 제네시스 파일을 업데이트하세요.
