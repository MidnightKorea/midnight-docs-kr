## Midnight Node 0.8.0 Release Notes

Midnight Node 0.8.0 릴리스입니다. 메트릭 향상, 성능 벤치마크, 키 유도 개선, testnet 환경 추가 지원이 포함되어 있습니다.


### New Features

- 핵심 원장에 Prometheus 메트릭 추가
- 핵심 Midnight 팔렛에 런타임 벤치마크 추가 (블록 계산 정확도 향상)
- 거버넌스 키 관리 개선
- 릴리스 노트 생성기 추가
- testnet-02 배포용 릴리스 설정 아티팩트 추가

### Changes

- Ledger 3.0.2에서 Ledger 3.0.5, 그리고 3.0.6으로 업그레이드했습니다.
- 트랜잭션 제너레이터의 버그를 수정했습니다.
- 향후 네트워크를 위한 키 유도 메커니즘을 업데이트했습니다.
- Rust 컴파일러를 1.83 버전으로 업그레이드했습니다.
- **성능:** 스토리지 캐싱에 대한 기본 설정을 추가했습니다.
- **성능:** 스토리지 종료에 대한 기본 설정을 업데이트했습니다.
- 사용성 향상을 위해 트랜잭션 제너레이터를 리팩토링했습니다.
