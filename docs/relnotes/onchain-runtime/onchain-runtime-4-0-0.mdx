# Onchain Runtime

Onchain Runtime은 영지식 증명 생성, 트랜잭션 검증, 프라이버시 보호 트랜잭션 처리를 가능하게 하는 Midnight 인프라 컴포넌트입니다.

자세한 내용은 [Onchain Runtime 튜토리얼](/api-reference/onchain-runtime/)을 참고하세요.

## Midnight 4.0.0 Onchain Runtime Release Notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight Onchain Runtime의 최신 업데이트와 개선 내용을 안내합니다.

### Stable Release

* **Version**: 4.0.0
* **Audience**: Midnight 생태계와 통합하는 개발자
* **Environment**: 프로덕션 준비 완료

이 릴리스(버전 `4.0.0`)는 안정적이며 프로덕션에 사용 가능합니다. 공개 및 프로덕션 환경에 안전하게 배포할 수 있습니다.

### Changelog

#### Breaking

- MPT 리프를 `Sp`로 저장할 수 있도록 필요한 객체를 `Storable`로 만들어 새로운 저장소 모델과 통합했습니다.
- Zswap 생성자에 segment ID를 추가했습니다. 실패 가능한 offer에는 `1`을, 보장된 offer에는 `0`을 설정해야 합니다.
- `ZswapLocalStateNoKeys`을 `ZswapLocalState`로 이름을 변경하고, 기존(키 포함) 상태를 제거했습니다.
- Pluto-Eris에서 BLS12-381로 전환했습니다.
- 직접적인 prover 키와 매개변수 대신 데이터 프로바이더를 사용하도록 전환했습니다.

#### Features

- Midnight용 키 자료를 가져오는 데이터 프로바이더를 추가했습니다. `MIDNIGHT_PARAM_SOURCE` 환경 변수로 소스를 재정의할 수 있습니다.
