# Proof Server

:::info Proof Server 릴리스는 이제 Ledger에 포함됩니다
Ledger 7.0.0부터 Proof Server 릴리스는 [Ledger 릴리스](./ledger)의 일부로 포함됩니다. 최신 Proof Server 릴리스와 업데이트는 [릴리스 노트 개요](./overview)를 참고하세요.
:::

Proof Server은 영지식 증명 생성, 트랜잭션 검증, 프라이버시 보호 트랜잭션 처리를 가능하게 하는 Midnight 인프라 컴포넌트입니다.

[관련 문서 링크](../guides/run-proof-server)

컴포넌트를 다운로드하려면 **Artifacts** 아래의 해당 링크를 클릭하세요.
