Proof Server은 영지식 증명 생성, 트랜잭션 검증, 프라이버시 보호 트랜잭션 처리를 가능하게 하는 Midnight 인프라 컴포넌트입니다.

## Midnight 3.0.7 Proof Server release notes

새로운 기능, 버그 수정, 호환성에 영향을 주는 변경 사항 등 Midnight Proof Server의 최신 업데이트와 개선 내용을 안내합니다.

## Changes

- proof server 로그에서 과도한 디버그 출력을 제거했습니다.
- 과도한 로깅으로 인한 proof server 성능 저하를 수정했습니다.
