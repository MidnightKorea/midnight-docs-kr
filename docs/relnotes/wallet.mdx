# Wallet SDK

Midnight Wallet SDK은 Midnight 블록체인을 위해 설계된 디지털 지갑의 SDK로, 사용자가 개인 키를 안전하게 보관하고, 자산을 관리하며, 탈중앙화 애플리케이션과 상호작용할 수 있게 합니다.
