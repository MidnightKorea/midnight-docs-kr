# Support

Midnight Testnet 참여자로서 여러분은 이슈 리포트, 개선 제안, 개발 과정에 직접 의견을 반영하여 Midnight의 미래를 함께 만들어갈 수 있습니다.

## Connectivity failures, bugs, general problems, and suggestions

문제 신고, 제안, 피드백, 도움 요청은 두 가지 방법으로 할 수 있습니다:

1. [Midnight Discord 서버](https://discord.gg/midnightnetwork)에서 논의에 참여하세요. Midnight 팀이 `#support` 채널을 정기적으로 모니터링하고 있습니다.
   **중요:** Midnight Discord 서버에 처음 참여하면 `#rules` 채널에서 규칙을 수락해야 다른 콘텐츠에 접근할 수 있습니다.
2. [Midnight 지원팀](mailto:support@midnight.network)에 이메일을 보내세요.

## Security vulnerabilities

Midnight 라이브러리나 인프라에서 보안 취약점을 발견한 경우, 아래 절차에 따라 신고해 주세요:

1. 보안 취약점을 Discord 지원 채널에 공개적으로 게시하지 **마세요**.
2. 발견한 내용을 Midnight 지원 이메일 주소 [support@midnight.network](mailto:support@midnight.network)로 보내주세요. 취약점에 대한 수정 방법이나 우회 방법이 있다면 함께 알려주세요.
3. Midnight 팀이 리포트를 검토한 뒤, 수정 사항을 배포하거나 취약점 완화 방법을 안내합니다. 결과는 직접 알려드립니다.
4. 별도 요청이 없는 한, 수정 사항이나 완화 방법 공개 시 발견자로서 크레딧을 표기합니다.
5. Midnight 팀이 수정 사항을 공개하거나 해당 취약점을 처리하지 않겠다고 알린 이후에는 자유롭게 공개할 수 있습니다.
