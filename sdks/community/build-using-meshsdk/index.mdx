# Build using MeshSDK

Mesh SDK를 사용하여 Midnight Network DApp을 구축하기 위한 전체 개발 설정입니다.

## Overview

**[@meshsdk/midnight-setup](https://www.npmjs.com/package/@meshsdk/midnight-setup)**은 Midnight Network 개발을 위한 종합 툴킷을 제공합니다.

이 툴킷에는 Midnight Network DApp을 구축하는 데 필요한 모든 것이 포함되어 있습니다:

- **@meshsdk/midnight-setup** - API와 타입을 제공하는 메인 npm 패키지
- **packages/ui** - React 예제 애플리케이션  
- **packages/cli** - 커맨드라인 도구
- **packages/api** - 핵심 API 구현

**예제 저장소** [midnight-setup](https://github.com/MeshJS/midnight-setup)

## Key features

- **Zero-knowledge privacy** - Midnight Network 전용으로 설계
- **TypeScript support** - 완전한 타입 안전성
- **React hooks** - 간편한 통합
- **Wallet integration** - Lace Beta Wallet 지원
- **CLI tools** - 개발 유틸리티


## Project structure

```
├── packages/
│   ├── api/          # Core API implementation
│   ├── ui/           # React example app
│   └── cli/          # Command-line tools
├── compact/          # Smart contract source
└── README.md
```
